1. Gr√°fico de Barras Apiladas con Facetado

Los gr√°ficos de barras apiladas permiten comparar m√∫ltiples categor√≠as dentro de una sola variable. Aqu√≠ observaremos la distribuci√≥n de pasajeros por clase (Pclass) y sexo (Sex), con un facetado adicional que indica si sobrevivieron o no.

ggplot(titanic, aes(x = Pclass, fill = Sex)) + 
  geom_bar(position = "stack") +
  labs(title = "Distribuci√≥n de Sexo por Supervivencia Clase de Pasajero", 
       x = "Clase de Pasajero", 
       y = "Cantidad de Pasajeros") +
  facet_wrap(~ Survived,labeller = as_labeller(c(`0`="No sobrevivi√≥",`1` ="Sobrevivio"))) + 
  theme_minimal()

Interpretaci√≥n: El facetado nos permite observar diferencias en la supervivencia seg√∫n el sexo y la clase de pasajero.

2. Mapa de Calor

El mapa de calor es √∫til para visualizar la relaci√≥n entre dos variables continuas. Aqu√≠ veremos c√≥mo la edad y el precio del pasaje se relacionan utilizando geom_bin2d.

ggplot(titanic, aes(x = Age, y = Fare)) + 
  geom_bin2d(bins = 30) + 
  scale_fill_gradient(low = "blue", high = "red") +
  labs(title = "Mapa de Calor: Relaci√≥n entre Edad y Precio del Pasaje", 
       x = "Edad", 
       y = "Precio del Pasaje") +
  theme_minimal()

Interpretaci√≥n: Las √°reas m√°s rojas indican una mayor densidad de observaciones. En este gr√°fico, se puede observar que la mayor√≠a de los pasajeros eran j√≥venes y pagaron precios de boletos m√°s bajos.

3. Gr√°fico de Densidad

Los gr√°ficos de densidad nos permiten observar la distribuci√≥n de una variable continua. En este caso, analizamos la distribuci√≥n de la edad seg√∫n si los pasajeros sobrevivieron o no.

ggplot(titanic, aes(x = Age, fill = factor(Survived))) + 
  geom_density(alpha = 0.5) +
  labs(title = "Distribuci√≥n de Edad seg√∫n Supervivencia", 
       x = "Edad", 
       fill = "Supervivencia") +
  scale_fill_manual(values = c("red", "green")) +
  theme_minimal()

Interpretaci√≥n: Los pasajeros que sobrevivieron eran generalmente m√°s j√≥venes que aquellos que no lo hicieron.

4. Gr√°fico de Dispersi√≥n y An√°lisis de Correlaci√≥n

Finalmente, utilizamos un gr√°fico de dispersi√≥n para ver la relaci√≥n entre la edad y el precio del pasaje, a√±adiendo una l√≠nea de tendencia con geom_smooth.

ggplot(titanic, aes(x = Age, y = Fare, color = factor(Survived))) + 
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) + 
  labs(title = "Relaci√≥n entre Edad y Precio del Pasaje", 
       x = "Edad", 
       y = "Precio del Pasaje",
       color = "Supervivencia") +
  scale_color_manual(values = c("red", "green")) +
  theme_minimal()
## `geom_smooth()` using formula = 'y ~ x'

Interpretaci√≥n: Podemos observar que no hay una fuerte correlaci√≥n entre la edad y el precio del pasaje. La l√≠nea de tendencia nos da una idea de la direcci√≥n general, pero la relaci√≥n parece ser d√©bil.
